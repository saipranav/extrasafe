How it works
  • Remember a good master password like (q1!Q)
  • Your master password + site name will be given to a Hash Algorithm
  • Say Master Password (q1!Q) in Site (facebook) will generate a output password (7{A32e08#Ef4b1f598@7)
  • So, what is a Hash Algorithm?
    • It will return the same output (a sequence of characters), when you give the same input.
    • It gives a result which cannot be reversed to get the input.
    • Hash Algorithm like SHA is popular and widely used even for bank transactions. Extrasafe Uses SHA3.
  • Extrasafe generates passwords on the go, we do not and will NEVER store any passwords in cloud or even in your computer.
Doubts that you may have     (hover to see the answers)
  • What if someone comes to know my master password?

    He must know that you are using Extrasafe to generate same site password.

    If you have changed the default Extrasafe options, he must know the exact options to generate the same site password.

  • What are default Extrasafe options?

    Extra security sequence : Its like another master password, also used to generate site passwords. Default :

    Start Index : Its a number which is used along with end index to derive a substring from the lengthy random string, which is the result from generation algorithm. Default : Minimum - 0, Maximum - 116

    End Index : Its a number which is used along with start index to derive a substring from the lengthy random string, which is the result from generation algorithm. Default : Minimum - 12, Maximum - 128

    To change default settings in browser extension you can goto manage extensions then select Options under the Extrasafe space.

    To change default settings in mobile application you can open menu and select Options.

  • What if something goes wrong?

    We have a backup plan!

    Extrasafe Portable is a web page where you can give your site name, master password, extrasafe options to generate the site password when ever you need.

    If you want Extrasafe to used offline you can download Extrasafe Portable
